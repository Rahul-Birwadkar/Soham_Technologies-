# Images Folder

## Current Setup
Images are loaded directly from Unsplash CDN in the HTML.
They display correctly in any browser.

## Replace With Real Photos
To use your own workshop/team photos:
1. Add your images to this folder (jpg/webp recommended)
2. Update the `src` attributes in `index.html`
3. Search for `images.unsplash.com` to find all image tags

## Recommended Real Photos to Take
- Workshop interior / team at work
- Close-up of PCB repair in progress  
- Servo drives being tested
- VFD/PLC panels before/after
- CNC machine maintenance
- Team photo (Mr. Gajanan sir + engineers)

## Image Format Tips
- Use JPG/WebP for photos
- Recommended size: 800×600px for cards, 1600×900px for hero
- Compress with TinyPNG.com before uploading
